Friends Day Pass for Youth 核对结论

可以包含游船。 SBB 官方写明，Friends Day Pass for Youth 的有效范围和 GA Travelcard 一样；而 SBB 对 GA 的说明包含 trains, S-Bahns, Postbuses, boats, mountain railways and urban public transport within the GA validity area。也就是说，在 GA 有效范围内的火车、巴士、城市交通和常规游船线路是包含的。

但你们这天最关键的 Grindelwald-First 并不是靠 Friends 通票就全包的项目。 Jungfrau 官方把 First Cable Car 和 Adventure Package 单独售卖；Adventure Package 页面还写了有 GA / Half Fare / Swiss Travel Pass reduced 的 reduced price。这说明 First 索道和山上娱乐项目需要另外买票，只是你们可能能拿到 reduced fare，而不是 Friends 一票全包。

你们这天若买 Friends，Olten ↔ Grindelwald 的公共交通段、以及如果你们之后去坐公营游船，基本是很值的；但 First 索道、Mountain Cart、Trottibike 请按山上产品单独买。

这天哪些通常不在 Friends 里全包

  • Grindelwald-First 索道：通常要单独买票。
  • Adventure activities：Mountain Cart、Trottibike、First Flyer、First Glider 属于山上娱乐项目，不是 Friends 通票直接全包。
  • 个别山地线路：就算在瑞士很多山地交通能享受折扣，也不等于 Friends 通票直接免费全包，最好按具体山头官方规则买。
